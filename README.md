# VisionExtract 🖼️
### AI-Based Image Isolation & Background Removal using Deep Learning Segmentation

An end-to-end AI application that automatically isolates subjects from images by removing backgrounds using deep learning-based segmentation pipelines.

## 🔍 What It Does

- Upload any image through the web interface
- Preprocesses and passes it through a deep learning segmentation model
- Isolates the subject by removing the background
- Generates a clean, downloadable output image
